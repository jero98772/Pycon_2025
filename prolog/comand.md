# 🐍🤖 Using PySWIP (Python + Prolog) on Linux (Manjaro & Ubuntu)

## 🛠️ 1. Install SWI-Prolog

### 📦 Ubuntu

```bash
sudo apt update
sudo apt install swi-prolog

```

### 📦 Manjaro

	sudo pacman -S swi-prolog


## Python

	python3 -m venv venv
	source venv/bin/activate
	pip install pyswip

	python3 <name>.py
